# 🤝 직원 채용

이 도구는 새로운 에이전트를 시스템에 채용(등록)하는 스크립트입니다.

## 파라미터 설명
- `agent_id`: 소문자 영문으로 된 고유 ID (예: translator)
- `name`: 사용자에게 보여질 직원의 이름 (예: 파파고)
- `role`: 직책 (예: 번역 전문가)
- `emoji`: 에이전트를 상징하는 이모지 (예: 🌐)
- `specialty`: 어떤 일을 잘하는지 짧게 요약. CEO가 보고 이 직원에게 일을 할당할지 결정하는 중요한 근거가 됩니다.

## 사용법
이 도구를 호출하면 시스템의 `custom_agents.json` 파일이 업데이트되며, 즉시 CEO의 워크플로우에 편입됩니다.
