# 구글 SEO 최적화 글 작성 도구

이 도구는 주어진 주제와 키워드를 바탕으로 구글 검색 엔진 최적화(SEO)에 유리한 글을 작성하는 데 도움을 줍니다. 글의 길이, 톤, 그리고 대상 독자를 명확히 지정하여 맞춤형 콘텐츠를 생성할 수 있습니다.

## 기능
- 특정 주제에 대한 SEO 친화적인 글의 개요 및 주요 내용 구성 시뮬레이션.
- 제공된 키워드를 자연스럽게 통합하여 검색 엔진 가시성을 높이는 전략 제시.
- 글의 목표 단어 수, 톤, 대상 독자를 고려하여 콘텐츠 방향 설정.

## 사용 방법
이 도구를 사용하여 SEO 최적화 글을 작성하려면 다음 매개변수를 제공해야 합니다.

### 매개변수
- **글 주제 (topic)**: 글의 주요 내용을 정의하는 핵심 주제입니다. (예: `2024년 최신 AI 트렌드`)
- **핵심 키워드 (keywords)**: 글에 반드시 포함되어야 할 SEO 키워드 목록입니다. 쉼표로 구분하여 여러 개를 입력할 수 있습니다. (예: `AI, 인공지능, 머신러닝`)
- **목표 단어 수 (word_count)**: 작성될 글의 대략적인 단어 수입니다. 최소 100단어 이상이어야 합니다. (예: `1000`)
- **글 톤 (tone)**: 글의 전반적인 분위기나 스타일을 결정합니다. 다음 중 하나를 선택할 수 있습니다: `전문적`, `친근한`, `설득력 있는`, `정보 제공`, `유머러스`. (예: `전문적`)
- **대상 독자 (target_audience)**: 글을 읽을 주된 독자층을 명시합니다. (예: `기술 블로거, 일반 대중, 마케터`)

### 예시
```json
{
  "topic": "구글 SEO 최적화 전략",
  "keywords": ["SEO 전략", "구글 랭킹", "검색 엔진 최적화"],
  "word_count": 1200,
  "tone": "전문적",
  "target_audience": "디지털 마케터"
}
```

위와 같이 매개변수를 설정하면, 도구는 '구글 SEO 최적화 전략'에 대한 전문적인 톤의 1200단어 분량의 글을 디지털 마케터를 대상으로 작성하며, 'SEO 전략', '구글 랭킹', '검색 엔진 최적화' 키워드를 포함하도록 시뮬레이션합니다. 실제 글 내용은 출력되지 않지만, 글의 구성과 SEO 적용 방안에 대한 가이드라인을 제공합니다.
