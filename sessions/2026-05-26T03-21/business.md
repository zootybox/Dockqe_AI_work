### 📊 키워드 수익성 점수(Profitability Score) 산출 공식 및 가중치 배분 설계

키워드의 수익성을 평가하기 위한 Profitability Score는 다음 세 가지 핵심 지표를 조합하여 산출합니다.

1.  **월간 검색량 (Monthly Search Volume, MSV):** 키워드의 잠재적 트래픽을 나타내는 지표. 높을수록 좋습니다.
2.  **예상 CPC (Cost Per Click):** 광고주가 해당 키워드에 지불하는 단가. 높을수록 애드센스 수익에 긍정적입니다.
3.  **경쟁 정도 (Competition Level):** 해당 키워드에 대한 경쟁 강도. 낮을수록 좋습니다 (높은 경쟁은 상위 노출 어려움).

#### **수익성 점수 (Profitability Score) 공식**

`Profitability Score = (w1 * 정규화된_MSV) + (w2 * 정규화된_CPC) - (w3 * 정규화된_Competition)`

여기서 `w1`, `w2`, `w3`는 각 지표의 가중치를 나타냅니다.

#### **지표 정규화 (Normalization)**

각 지표는 측정 단위와 범위가 다르므로, 0에서 1 사이의 값으로 정규화하여 가중치 적용 및 비교가 용이하도록 합니다.

*   **정규화된_MSV (Normalized MSV):**
    `정규화된_MSV = (MSV - 최소_MSV) / (최대_MSV - 최소_MSV)`
*   **정규화된_CPC (Normalized CPC):**
    `정규화된_CPC = (CPC - 최소_CPC) / (최대_CPC - 최소_CPC)`
*   **정규화된_Competition (Normalized Competition):**
    경쟁 정도는 일반적으로 0~1 사이의 값으로 제공되거나, '낮음', '중간', '높음' 등으로 구분됩니다.
    *   숫자 값인 경우: `정규화된_Competition = Competition`
    *   범주형 값인 경우: '낮음' = 0.2, '중간' = 0.5, '높음' = 0.8 등으로 매핑 후 정규화하거나, `(Competition_Index - 최소_Competition_Index) / (최대_Competition_Index - 최소_Competition_Index)` 형태로 변환합니다. 여기서는 0~1 사이의 숫자 값으로 가정하고 정규화가 필요 없다고 판단합니다. (높을수록 경쟁이 심하다는 의미이므로, 점수 산출 시 마이너스 가중치를 적용하여 수익성을 낮춤)

#### **가중치 (Weights) 배분**

각 지표의 중요도를 반영하여 가중치를 설정합니다. 초기 가중치는 다음과 같이 제안합니다. (총합 100%를 기준으로 비율 설정)

*   `w1` (월간 검색량): **40%**
    *   이유: 아무리 CPC가 높아도 검색량이 없으면 수익이 발생하기 어렵습니다. 잠재적 트래픽 확보가 중요합니다.
*   `w2` (예상 CPC): **40%**
    *   이유: 애드센스 수익의 핵심 요소입니다. 높은 CPC는 적은 클릭으로도 높은 수익을 가능하게 합니다.
*   `w3` (경쟁 정도): **20%**
    *   이유: 경쟁이 심하면 상위 노출이 어려워 트래픽 확보가 힘들어집니다. 하지만 MSV와 CPC에 비해 직접적인 수익 기여도는 낮으므로 상대적으로 낮은 가중치를 부여합니다.

**초기 가중치 제안:** `w1 = 0.4`, `w2 = 0.4`, `w3 = 0.2`

#### **키워드 등급 (S~D) 분류 기준**

산출된 Profitability Score를 기반으로 키워드를 S, A, B, C, D 등급으로 분류합니다. 등급 경계는 데이터 분포에 따라 조정될 수 있으나, 초기에는 다음과 같이 제안합니다. (Score는 0~1 사이의 값으로 정규화된 후 가중치 적용 시 최종적으로 0~1 사이의 값으로 나옴)

*   **S 등급 (매우 우수):** Score >= 0.8
    *   높은 검색량, 높은 CPC, 낮은 경쟁도를 모두 만족하는 최상위 키워드. 즉시 콘텐츠 생성 및 집중 투자가 필요합니다.
*   **A 등급 (우수):** 0.6 <= Score < 0.8
    *   전반적으로 우수한 지표를 가진 키워드. S 등급 다음으로 우선순위를 가지고 콘텐츠를 생성합니다.
*   **B 등급 (보통):** 0.4 <= Score < 0.6
    *   평균적인 수익성을 가진 키워드. 특정 지표가 부족하지만 다른 지표로 보완 가능한 경우.
*   **C 등급 (미흡):** 0.2 <= Score < 0.4
    *   수익성이 낮은 키워드. 트래픽 확보 또는 수익 창출이 어려울 수 있으므로 신중한 접근이 필요합니다.
*   **D 등급 (매우 미흡):** Score < 0.2
    *   매우 낮은 수익성을 가진 키워드. 특별한 전략적 목적이 없는 한 콘텐츠 생성에서 제외합니다.

#### **추가 고려사항:**

*   **최소 임계값:** MSV 또는 CPC가 특정 값 이하인 키워드는 Profitability Score가 높아도 D 등급으로 강등하는 규칙을 추가할 수 있습니다 (예: MSV < 100, CPC < 100원).
*   **데이터 출처:** 각 지표의 데이터 출처 (Google Keyword Planner, Ahrefs, SEMrush 등)를 명확히 하고, 데이터의 신뢰성을 확보하는 것이 중요합니다.
*   **A/B 테스트 및 최적화:** 실제 운영 데이터를 통해 가중치 및 등급 분류 기준을 지속적으로 A/B 테스트하고 최적화해야 합니다.
*   **트렌드 반영:** 키워드의 계절성, 시의성 등 트렌드를 반영하여 점수를 조정하는 로직을 추가할 수 있습니다.

이 공식과 가중치, 등급 분류 기준은 초기 제안이며, 실제 키워드 데이터에 적용해보면서 지속적으로 개선해나가야 합니다.
