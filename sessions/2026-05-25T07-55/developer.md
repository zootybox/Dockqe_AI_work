## 🏗️ 워드프레스 자동 포스팅 시스템 아키텍처 설계 및 기술 스택

워드프레스 REST API를 활용한 자동 포스팅 시스템은 크게 다음과 같은 모듈로 구성됩니다. 각 모듈은 독립적으로 기능하며, 서로 유기적으로 연동되어 자동화된 글 발행 프로세스를 완성합니다.

### 1. ⚙️ 키워드 분석 및 제목 생성 모듈
*   **기능**: 고수익 키워드 발굴, 검색량 및 경쟁도 분석, SEO 최적화된 제목 생성.
*   **기술 스택**:
    *   **Python**: 데이터 분석 라이브러리 (Pandas, NumPy), 웹 스크래핑 (BeautifulSoup, Scrapy), 키워드 분석 API 연동 (Google Keyword Planner API 또는 유사 서비스).
    *   **AI/ML (선택 사항)**: 제목 생성 및 키워드 조합 최적화를 위한 LLM (Large Language Model) 연동 (예: OpenAI GPT API, Google Gemini API).
*   **연동**: 다음 모듈(글 생성 모듈)로 키워드와 생성된 제목을 전달.

### 2. 📝 글 자동 생성 모듈
*   **기능**: 키워드와 제목을 기반으로 워드프레스 포스팅에 적합한 본문 콘텐츠 자동 생성. Writer 에이전트의 가이드라인(애드센스 친화적 블로그 글 구조, 카피라이팅 템플릿)을 반영.
*   **기술 스택**:
    *   **Python**: 텍스트 생성 및 가공.
    *   **AI/LLM**: GPT-3.5/4, Gemini 등 LLM API를 활용하여 고품질의 자연스러운 글 생성.
    *   **콘텐츠 구조화**: 마크다운 또는 HTML 형식으로 글 구조화 로직 구현.
*   **연동**: 생성된 본문 콘텐츠를 이미지 삽입 및 광고 배치 모듈로 전달.

### 3. 🖼️ 이미지 자동 삽입 모듈
*   **기능**: 글 내용과 관련된 고품질 이미지를 자동으로 검색, 다운로드 및 워드프레스 미디어 라이브러리에 업로드 후 본문에 삽입. 이미지 SEO (alt 태그, 파일명) 최적화.
*   **기술 스택**:
    *   **Python**: 이미지 검색 API 연동 (Unsplash API, Pexels API 등), 이미지 처리 라이브러리 (Pillow).
    *   **워드프레스 REST API**: 미디어 업로드 및 본문 이미지 삽입.
*   **연동**: 본문에 이미지가 삽입된 형태로 다음 모듈(애드센스 광고 배치 모듈)로 전달.

### 4. 💰 애드센스 광고 최적 배치 모듈
*   **기능**: PublishFlow 광고 배치 표준과 Writer 에이전트의 가이드라인(H2 태그 아래, 글 중간, 결론 전 등)에 따라 생성된 글 본문에 애드센스 광고 코드를 최적의 위치에 삽입.
*   **기술 스택**:
    *   **Python**: HTML 파싱 및 조작 (BeautifulSoup), 정규표현식을 활용한 광고 삽입 로직.
    *   **워드프레스 Ad Inserter 연동 (선택 사항)**: 워드프레스 플러그인 Ad Inserter의 블록 번호를 활용하여 광고 슬롯 ID와 연동.
*   **연동**: 최종적으로 광고가 삽입된 HTML 형태의 글을 워드프레스 포스팅 모듈로 전달.

### 5. 🚀 워드프레스 포스팅 모듈
*   **기능**: 최종적으로 준비된 제목, 본문(이미지, 광고 포함), 카테고리, 태그 등을 워드프레스 REST API를 통해 포스팅.
*   **기술 스택**:
    *   **Python**: `requests` 라이브러리를 활용하여 워드프레스 REST API 연동.
    *   **워드프레스 REST API**: `/wp/v2/posts` 엔드포인트를 사용하여 글 생성 및 발행.
*   **연동**: 워드프레스 사이트.

### 6. 🌐 배포 및 실행 환경
*   **기능**: 전체 자동화 시스템을 안정적으로 실행하고 관리.
*   **기술 스택**:
    *   **Cloudflare Pages/Workers (선택 사항)**: 서버리스 환경에서 특정 모듈(예: 키워드 분석, 글 생성)을 실행하거나 API 게이트웨이 역할.
    *   **Google Cloud Functions / AWS Lambda (선택 사항)**: Python 기반의 서버리스 함수로 각 모듈을 배포하여 비용 효율적인 운영.
    *   **Cron Job / 스케줄러**: 정기적인 자동 포스팅 실행을 위한 스케줄링.
    *   **Docker**: 각 모듈을 컨테이너화하여 배포 및 관리 용이성 확보.
*   **모니터링**: 에러 로깅, 포스팅 성공/실패 알림 (예: 텔레그램 봇 연동).

## 📝 개발 계획

다음은 위의 아키텍처를 기반으로 한 단계별 개발 계획입니다.

### 1단계: 핵심 기능 프로토타입 개발 (2주)
1.  **워드프레스 REST API 연동 확인**: `requests` 라이브러리를 사용하여 워드프레스에 간단한 테스트 글을 발행하는 Python 스크립트 작성. (인증 방식 확인: Application Passwords 권장)
    *   **검증**: 실제 워드프레스에 글이 정상적으로 발행되는지 확인.
2.  **글 자동 생성 (LLM 연동)**: 키워드와 제목을 입력받아 LLM API (예: OpenAI GPT-3.5)를 통해 간단한 본문 텍스트를 생성하는 프로토타입 개발.
    *   **검증**: 생성된 글의 품질 및 관련성 평가.
3.  **기본적인 광고 배치 로직**: 생성된 글 본문에 특정 마커(예: `[AD_SLOT_1]`)를 기준으로 애드센스 광고 코드를 삽입하는 로직 구현.
    *   **검증**: 광고 코드가 HTML에 올바르게 삽입되는지 확인.

### 2단계: 이미지 자동 삽입 및 광고 최적화 (2주)
1.  **이미지 검색 및 삽입**: Unsplash API 등을 활용하여 키워드에 맞는 이미지를 검색하고, 워드프레스 미디어 라이브러리에 업로드 후 본문 HTML에 삽입하는 로직 구현. `alt` 태그 자동 생성 포함.
    *   **검증**: 이미지가 정상적으로 삽입되고 `alt` 태그가 SEO 친화적인지 확인.
2.  **애드센스 광고 최적 배치 로직 고도화**: Writer 에이전트의 가이드라인(H2 태그 아래, 글 중간, 결론 전 등)을 기반으로 HTML 파싱 및 구조 분석을 통해 광고를 지능적으로 배치하는 로직 개발. PublishFlow의 5개 광고 슬롯 ID와 Ad Inserter 블록 연동 고려.
    *   **검증**: 광고가 의도된 위치에 정확히 삽입되는지, 사용자 경험을 해치지 않는지 수동 검증.

### 3단계: 키워드 분석 및 시스템 통합 (2주)
1.  **키워드 분석 및 제목 생성 모듈 개발**: Google Keyword Planner API 등을 활용하여 고수익 키워드를 발굴하고, 이를 기반으로 SEO 친화적인 제목을 생성하는 로직 구현. (Researcher 에이전트의 결과물 연동)
    *   **검증**: 생성된 키워드 및 제목의 품질과 수익성 잠재력 평가.
2.  **전체 시스템 통합**: 각 모듈을 연결하여 키워드 발굴부터 워드프레스 포스팅까지의 전체 자동화 파이프라인 구축.
    *   **검증**: 엔드-투-엔드 테스트를 통해 전체 시스템의 정상 작동 확인.

### 4단계: 배포, 모니터링 및 최적화 (1주)
1.  **배포 환경 구축**: Google Cloud Functions 또는 Docker를 활용하여 시스템을 배포하고 Cron Job을 통해 스케줄링.
2.  **모니터링 및 알림**: 포스팅 성공/실패 여부, 에러 발생 시 텔레그램 봇을 통한 알림 시스템 구축.
3.  **성능 최적화**: 시스템의 처리 속도, API 호출 비용 등을 고려하여 코드 최적화.
4.  **A/B 테스트 및 개선**: 광고 배치 로직, 글 생성 방식 등에 대한 A/B 테스트를 통해 지속적인 수익 최적화.
