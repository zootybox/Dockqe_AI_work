### **2. 황금 키워드 수집 방법**

로컬 환경에서 키워드 데이터를 수집하는 구체적인 방법은 다음과 같습니다.

*   **API 활용 (Python):**
    *   `requests` 라이브러리를 사용하여 네이버 데이터랩과 같은 공공 API에 요청을 보내고 JSON 형태의 데이터를 수집합니다.
    *   API 문서에 명시된 인증 절차(API 키, Secret 등)를 따라야 합니다.
*   **웹 스크래핑 (Python):**
    *   `requests` 라이브러리로 웹 페이지의 HTML 콘텐츠를 가져옵니다.
    *   `BeautifulSoup` 또는 `lxml` 라이브러리를 사용하여 HTML을 파싱하고 필요한 텍스트 데이터(예: 제목, 본문 내용, 태그, 검색어 순위 등)를 추출합니다.
    *   동적으로 로딩되는 콘텐츠(JavaScript 기반)의 경우 `Selenium` 라이브러리를 활용하여 웹 브라우저를 제어하며 데이터를 수집할 수 있습니다.
    *   **스크래핑 시 유의사항:**
        *   대상 웹사이트의 `robots.txt` 파일을 확인하여 스크래핑 허용 여부 및 제한 사항을 준수합니다.
        *   IP 차단 방지를 위해 `User-Agent`를 변경하거나, 필요한 경우 프록시 서버를 활용하는 방안을 고려합니다.
        *   서버에 과부하를 주지 않도록 적절한 지연 시간(delay)을 설정하여 요청 빈도를 조절합니다.

### **3. 수집된 키워드 데이터의 정제 및 가공 절차**

수집된 원시 데이터는 다음과 같은 절차를 거쳐 '황금 키워드'로 가공됩니다.

*   **1단계: 초기 필터링 및 클리닝**
    *   **불필요한 문자 제거:** 특수문자, HTML 태그, 광고성 문구 등 키워드 분석에 방해가 되는 요소를 제거합니다.
    *   **언어 필터링:** 한국어 키워드에 집중하기 위해 다른 언어(영어, 중국어 등) 키워드를 필터링합니다.
    *   **길이 필터링:** 너무 짧거나(예: 1~2글자) 너무 긴 키워드를 제거하여 분석의 효율성을 높입니다.
    *   **오탈자 및 띄어쓰기 정규화:** 자주 발생하는 오탈자를 수정하고, 띄어쓰기 규칙을 통일하여 키워드 유사도를 높입니다.
*   **2단계: 중복 제거 및 유사 키워드 그룹화**
    *   **중복 제거:** 완전히 동일한 키워드를 제거합니다.
    *   **유사 키워드 그룹화:** 의미적으로 유사하거나 동의어 관계에 있는 키워드들을 그룹화하여 분석의 정확도를 높이고, '롱테일 키워드' 발굴의 기반을 마련합니다. (예: "건강 보조제"와 "건강기능식품")
*   **3단계: 메타데이터 추가 및 수익성 평가**
    *   **검색량 추정:** 네이버 데이터랩의 트렌드 지수, 구글 트렌드 데이터 등을 활용하여 키워드의 상대적인 검색량 변화 추이를 파악하고, 월별 검색량 1,000회 이상인 키워드를 우선 선별합니다. [근거: 회사 의사결정 로그, 추측]
    *   **경쟁도 추정:** 검색 결과 페이지 수, 상위 노출 웹사이트의 도메인 권위 등을 기반으로 키워드의 경쟁도를 대략적으로 추정합니다. [근거: 추측]
    *   **CPC (Cost Per Click) 추정:** 직접적인 CPC 데이터는 유료 툴이 필요하나, 관련 광고의 노출 여부, 키워드의 상업적 의도 등을 통해 예상 입찰가 1,000원 이상인 키워드를 필터링합니다. [근거: 회사 의사결정 로그, 추측]
    *   **수익성 점수 산출 및 등급 분류:** 현빈 에이전트가 설계한 '수익성 점수' 공식을 적용하여 키워드의 가치를 수치화하고, S~D 등급으로 분류합니다. [근거: 통합 스케줄] 이 점수를 기준으로 최종 '황금 키워드' 리스트를 선정합니다.
*   **4단계: 키워드 확장 및 카테고리 분류**
    *   **연관 키워드 확장:** 수집된 핵심 키워드를 기반으로 구글/네이버의 자동 완성, 연관 검색어 기능을 활용하여 추가 키워드를 발굴합니다.
    *   **카테고리 분류:** 수집된 키워드를 주제별(예: 복지, 금융, IT, 건강 등), 산업별로 분류하여 블로그 콘텐츠 기획에 용이하도록 구조화합니다.
