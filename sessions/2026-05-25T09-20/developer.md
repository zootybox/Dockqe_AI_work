*   **주요 기능 확정:** 키워드 수집, 분석, 시각화 기능을 핵심으로 확정했습니다.

**다음 개발 단계:**

1.  **백엔드 API 개발 착수:** FastAPI를 이용해 키워드 수집, 분석, 데이터 저장을 위한 API 엔드포인트들을 구현할 예정입니다.
