# 📋 작업 브리프

**원 명령:** 워드프레스를 글을 전부 분석한거야? 분석한 글의 갯수와 문제가 있는것 분류하고 제목을 알려 줘야지?

## 요약
워드프레스 블로그 글 전체 분석 여부 확인 및 문제 있는 글 목록 요청

## 분배
- **💻 코다리**: 워드프레스 http://mapbogi.com에서 추출한 글의 총 개수를 확인하고, 추출 과정에서 발생한 문제점(예: 누락된 글, 오류)이 있는지 보고합니다. 이전에 추출된 데이터가 있다면 해당 데이터의 글 개수도 함께 보고합니다.
- **✍️ Writer**: developer 에이전트로부터 전달받은 워드프레스 글 데이터(혹은 기존에 분석한 데이터)를 기반으로 '가치 없는 콘텐츠'로 분류될 수 있는 글의 기준을 명확히 제시하고, 해당 기준에 부합하는 글의 제목과 그 이유를 구체적으로 분류하여 목록화합니다.
- **📱 영숙**: developer 에이전트와 writer 에이전트의 분석 결과를 종합하여, 워드프레스 글의 총 개수, 문제 있는 글의 분류 기준, 해당 글들의 제목 및 문제점을 CEO에게 보고합니다.
