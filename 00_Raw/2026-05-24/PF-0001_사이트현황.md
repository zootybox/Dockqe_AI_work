# 🏢 PublishFlow 사이트 운영 현황

## 📌 한 줄 통찰 (Agent Directive)
> PublishFlow는 3개 사이트를 운영하는 1인 AdSense 수익화 회사다. 코드 작성 및 포스팅 시 항상 이 사이트 표준을 따를 것.

## 📖 핵심 프롬프트
- **Role:** PublishFlow 전담 개발자 및 콘텐츠 전략가
- **Constraint 1:** 모든 웹 유틸리티 도구 코드는 modutools 표준(tool- 접두사, mt-header/footer) 준수
- **Constraint 2:** 모든 블로그 글은 Ad Inserter 숏코드 5개 포함 및 SEO 준수

## 📊 사이트별 현황

### zootybox.com (WordPress)
- **콘텐츠:** 정부 복지 혜택 정보 자동 포스팅
- **엔진:** Apps Script + Gemini API + WordPress REST API 자동 발행 파이프라인
- **수익화:** Google AdSense (Ad Inserter 블록 1-11 사용)
- **특이사항:** 태그 아카이브 페이지는 robots.txt를 통해 검색 엔진 크롤링 차단하여 크롤 예산 최적화.

### mapbogi.com (WordPress)
- **콘텐츠:** 전국 주요 여행지 및 맛집 등 지도 기반 정보 서비스
- **엔진:** Make 자동화 (9개 모듈) 연계 연동
- **수익화:** Google AdSense
- **특이사항:** Featured Image Module 16을 이용한 썸네일 자동 생성 연동.

### modutools.com (Cloudflare Pages)
- **콘텐츠:** 40개 웹 유틸리티 도구 서비스
- **도구 카테고리:** 계산기(9), 이미지(7), 텍스트(5), 변환기(10), 생성기(4), 생활(4), 운동(1)
- **수익화:** Google AdSense
- **디자인 가이드:** Pretendard 폰트, 라이트 테마, 둥근 카드 UI
