# 📊 Google AdSense 광고 배치 표준

## 📌 한 줄 통찰 (Agent Directive)
> 광고 슬롯 배치와 자바스크립트 광고 코드 보호 표준을 준수하여 AdSense 노출 및 수익을 최적화한다.

## 🛠 광고 슬롯 표준 (전 사이트 공통)

### 1. 주요 광고 위치 매핑
- **ad-slot-top:** 상단 광고 슬롯 → Ad Inserter Block 1에 해당
- **ad-slot-action:** 행동 유도 버튼 주변 슬롯 → Ad Inserter Block 2에 해당
- **ad-slot-mid:** 본문 중간 슬롯 → Ad Inserter Block 3에 해당
- **ad-slot-bottom:** 본문 최하단 슬롯 → Ad Inserter Block 5에 해당

### 2. WordPress 포스팅 주의사항
- 모든 글 작성 시 최상단에 `[adinserter disable="7,8,9,10,11"]` 숏코드를 수동으로 반드시 삽입하여 사이드바나 푸터 광고 등 특정 레이아웃 광고 블록과의 충돌을 방지합니다.

## 🔐 JavaScript 삽입 및 보호 표준

웹 도구(modutools) 등에 광고나 로직 코드를 삽입할 때 코드 복제 및 애드블록 우회를 방지하기 위해 다음 방식을 사용합니다:
- **Base64 인코딩 적용:** 스크립트 내 광고 인젝션이나 크롤링 탐지 코드는 `eval(atob("..."))` 형태로 압축/인코딩하여 삽입합니다.
- **한글 및 특수문자 보호:** 한글 텍스트 유니코드 이스케이프(`\uXXXX`) 처리 후 인코딩을 적용합니다.
